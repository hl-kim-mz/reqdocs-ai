# 서비스 개요 (Service Overview)

> v1.0 | 작성일: 2026-04-09

---

## 1. 서비스 정의

**ReqDocs AI**는 비정형화된 요구사항·제안사항을 입력받아,  
Claude AI가 분석하여 개발에 필요한 정형 문서를 자동 생성하는 웹 서비스입니다.

### 핵심 가치

```
비정형 입력 (회의록, 이메일, 메모, 인터뷰 노트)
        ↓
    AI 분석·구조화
        ↓
정형 개발 문서 자동 생성
(PRD · 기능 목록 · 기능 명세서 · API 문서 · ERD)
```

---

## 2. 생성 문서 종류

| 문서 | 설명 | 형식 |
|------|------|------|
| PRD | 제품 요구사항 정의서 | Markdown |
| 기능 목록 | Feature List + MoSCoW 우선순위 | 테이블 |
| 기능 명세서 | 기능별 상세 정의 (조건·예외 포함) | Markdown |
| API 문서 | Endpoint 목록 + Request/Response | OpenAPI 3.0 |
| ERD | 엔티티 관계 다이어그램 | Mermaid |

---

## 3. 지원 입력 형식

- 텍스트 직접 입력 (최대 10,000자 권장)
- 파일 업로드: `.txt` `.md` `.pdf` `.docx`
- (Post-MVP) URL 입력: Notion, Google Docs 링크

---

## 4. 사용자 시나리오

### 주요 페르소나
- **서비스 기획자**: 고객 미팅 후 회의록을 붙여넣어 PRD 초안 즉시 생성
- **개발 PM**: 기능 요청사항을 입력해 API 문서·ERD 초안 확보
- **스타트업 창업자**: 아이디어 메모를 개발팀이 읽을 수 있는 문서로 변환

### 핵심 플로우

```
1. 요구사항 텍스트 입력 (또는 파일 업로드)
2. 생성할 문서 종류 선택
3. [문서 생성] 클릭 → 스트리밍으로 실시간 생성 확인
4. 탭 전환으로 각 문서 확인 및 편집
5. Markdown / PDF / DOCX 형식으로 내보내기
```

## 5. MVP 범위

### MVP (빠른 검증, 2~3주)

| 항목 | 포함 여부 | 비고 |
|------|-----------|------|
| 텍스트 입력 | ✅ | |
| 파일 업로드 | ❌ | Post-MVP |
| PRD 생성 | ✅ | |
| 기능 목록 생성 | ✅ | |
| 기능 명세서 생성 | ❌ | Post-MVP |
| API 문서 생성 | ❌ | Post-MVP |
| ERD 생성 | ❌ | Post-MVP |
| 프로젝트 저장 | ❌ | 세션 내 임시 저장만 |
| 로그인/회원가입 | ❌ | Post-MVP |
| 내보내기 (MD) | ✅ | 복사 버튼 |

### Post-MVP

- 파일 업로드 지원 (PDF, DOCX)
- 기능 명세서 + API 문서 + ERD 생성 추가
- 프로젝트 저장 및 이력 관리
- 회원가입 / 로그인
- Notion / Confluence 연동 내보내기
- 문서 인라인 편집 기능

## 6. 성공 지표

| 지표 | 목표 |
|------|------|
| PRD 생성 정확도 | 도메인 일치율 > 80% |
| 문서 생성 시간 | 일반 요구사항 기준 < 30초 |
| 파일 업로드 지원 | PDF / DOCX / TXT (최대 10MB) |
| API 응답 시간 | 문서 조회 < 200ms |
