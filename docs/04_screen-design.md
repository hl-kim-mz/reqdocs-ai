# 화면 설계 (Screen Design)

> v1.0 | 작성일: 2026-04-09  
> 기준: Desktop-first | 최소 해상도: 1280px | 최적: 1440px~1920px

---

## 1. 정보구조도 (IA)

```
🌐 ReqDocs AI (웹 서비스)
│
├── / (랜딩 페이지)
│
├── /auth/login
│
└── /workspace  (로그인 후 — 사이드바 고정)
    ├── /workspace/dashboard     대시보드 (프로젝트 목록)
    ├── /workspace/new           새 프로젝트 생성
    └── /workspace/[projectId]   결과 문서 뷰어
        ├── /prd
        ├── /features
        ├── /spec
        ├── /api
        └── /erd
```

---

## 2. 레이아웃 시스템

```
┌──────────────────────────────────────────────────────────────┐
│  TOPBAR  (64px 고정)                                         │
├──────────┬───────────────────────────────────────────────────┤
│          │                                                   │
│ SIDEBAR  │               MAIN CONTENT AREA                  │
│ (240px   │               (flex-grow)                        │
│  고정)   │                                                   │
│          │                                                   │
└──────────┴───────────────────────────────────────────────────┘
```

| 해상도 | Sidebar | 문서 영역 | 사이드 패널 | 여백 |
|--------|---------|-----------|-------------|------|
| 1280px (최소) | 240px | 720px | 240px | 80px |
| 1440px (최적) | 240px | 820px | 280px | 100px |
| 1920px (와이드) | 240px | 1100px | 320px | 260px |

- 사이드바 접기: 240px ↔ 64px (아이콘 모드)

---

## 3. 사용자 플로우 (Happy Path)

```
[랜딩] → [시작하기 CTA]
    → (비로그인) → [로그인]
    → [/workspace/new — 입력 화면]
        → 텍스트 입력 or 파일 업로드
        → 생성할 문서 종류 선택
        → [문서 생성 클릭]
        → [우측 패널: 스트리밍 진행 상태]
        → [/workspace/[id] — 결과 화면]
            → 탭 전환 (PRD / 기능 / 명세 / API / ERD)
            → 편집 / 다운로드 / 공유
```

**인터랙션**:
- 파일 업로드 → 텍스트 추출 후 textarea에 자동 삽입
- 텍스트 미입력 시 `문서 생성` 버튼 비활성화
- 문서 종류 전체 해제 시 버튼 비활성화
- 생성 클릭 → 우측 패널이 진행 상태로 전환 (페이지 이동 없음)
